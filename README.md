# 📚 Library Management System

A full-stack **Library Management System** built to manage books, users, and library operations efficiently.  
This project provides an easy-to-use interface for handling book records, user management, issuing/returning books, and tracking availability.

## 🚀 Features

- 📖 Add, update, delete, and view books
- 👤 Manage users / members
- 🔍 Search books by title, author, or category
- 📚 Issue and return books
- 📅 Track availability and due dates
- 🛠️ Admin-friendly dashboard for library operations
- 🌐 Full-stack architecture with separate frontend and backend

---
